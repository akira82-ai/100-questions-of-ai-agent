## 版本与数据截止说明

- 本书内容基于 lark-cli v1.0.68（2026-07-09）及之前版本的公开信息撰写
- GitHub 数据截止 2026-07-10：15.4k stars、69 releases、159 open issues
- 三家 CLI 对比数据截止 2026-07-10
- 飞书 CLI 迭代频率很高（平均每周 2-3 次版本更新），部分命令参数、Skill 列表、错误码可能在新版本中有变化，以官方最新文档为准

## 如何反馈

- 本书内容错误或过时：在 GitHub 仓库提 Issue
- 飞书 CLI 本身的 bug：在 larksuite/cli 仓库提 Issue
- 飞书开放平台 API 问题：查阅飞书开放平台官方文档或提交工单
